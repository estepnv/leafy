# Leafy [![Build Status](https://travis-ci.org/estepnv/leafy.svg?branch=master)](https://travis-ci.org/estepnv/leafy)

A ruby toolkit for dynamic custom attributes.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'leafy'
```

And then execute:

    $ bundle
